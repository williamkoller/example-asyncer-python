# Example Asyncer Python

## Required Python 3 and `python3.10-venv`

## Install

```bash
sudo apt install python3.10-venv
```

## Structure environment with venv

```bash
python3.10 -m venv .env
```

## Activate environment

```bash
source .env/bin/activate
```

## Execute code

```bash
python3.10 main.py
```
